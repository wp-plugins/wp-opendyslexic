== Description ==

This plugin allows registered users to enable the OpenDyslexic font (see: http://www.opendyslexic.org/) on your Wordpress site.
This free font is designed to help people with dyslexia read more easily.
After activation this plugin adds an extra option to the profile page, where users can choose to use the OpenDyslexic font on
the website and/or within the admin, or not at all.

== Installation ==

0. Download and unzip
1. Upload the folder from the plugin to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Do you support other languages? =

So far this plugins is only available English and Dutch are included. You are free to create a translation for your language.
The required .POT file is in the 'languages' folder.

= Will the OpenDyslexic font be visible to all visitors? =

The font is only visible for the registered users who have enabled the font on their user profile. All other users and visitors will see the regular font as defined by your theme/CSS.

== Known bugs ==

* Some of the icons from the admin menu don't show because those symbols are not (yet) included in the OpenDyslexic font.
